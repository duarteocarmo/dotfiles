# NobodyWho

NobodyWho is a local-first inference engine for running GGUF/open-weight LLMs inside apps.

## Workflow

1. Identify the target platform and task.
2. Read the relevant current docs page(s), using `llms.txt` to discover pages.
3. If writing code, verify API names and installation steps from the docs or installed package/types.
4. Prefer links to source docs over copying volatile details into the answer.

## Stable reminders

- NobodyWho runs local GGUF models; it is not a hosted LLM API.
- Chat, embeddings, reranking/cross-encoding, multimodal, tool calling, and structured output may need different model capabilities.
- Remote model loading and cache behavior can vary by platform; check the platform docs.
- For private/gated Hugging Face models, check the docs for the current authentication pattern.
